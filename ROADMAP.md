# Roadmap — Web3 Dive

## Milestone 1: Foundations

### 1. Bitcoin Whitepaper
Read the original whitepaper by Satoshi Nakamoto.

- Proof-of-Work consensus
- Decentralized ledger
- UTXO model
- Peer-to-peer electronic cash

### 2. Ethereum.org Course (es)
Complete the course at [ethereum.org/es](https://ethereum.org/es/).

- Ethereum fundamentals
- Smart contracts
- EVM (Ethereum Virtual Machine)
- dApps and wallets

### 3. The Internet of Money — Andreas Antonopoulos
Read the book. Focus on the vision and socio-economic impact of Bitcoin.

- Money and decentralization
- Trust models
- Bitcoin's role in society

---

## Milestone 2: DeFi & Smart Contracts

### 1. Uniswap — Research
Understand how Uniswap works.

- AMM (Automated Market Maker)
- Liquidity pools
- Constant product formula: $x \cdot y = k$
- Impermanent loss

### 2. DeFi Protocols (Aave, Compound)
Explore lending and borrowing protocols.

- Lending / Borrowing mechanics
- Interest rate models
- Liquidations and collateral

### 3. Join Project Discords
Join active communities for continuous learning.

- Follow discussions
- Ask questions
- Stay up to date with ecosystem news

### 4. Cryptozombies.io — Solidity Course
Start the interactive Solidity course.

- Solidity syntax and structure
- Mappings, structs, arrays
- Contract inheritance
- Gas optimization basics

---

## Milestone 3: Hands-On & Community

### 1. Deploy First Smart Contract (Remix + Sepolia)
Write, compile and deploy a simple smart contract on Sepolia testnet.

- Use Remix IDE
- Configure MetaMask for Sepolia
- Interact with the contract on Etherscan

### 2. Contribute to an Open Source Project
Find a web3 repo with `good-first-issue` labels.

- Set up the dev environment
- Make a meaningful PR
- Learn from code reviews

### 3. Attend In-Person Events
Ethereum Lima meetups and local web3 events.

- Network with builders
- Learn from talks and workshops
- Find potential collaborators

### 4. Define My Web3 Approach
Establish a clear profile as a **developer**.

- Choose a tech stack (Solidity, Hardhat/Foundry, frontend)
- Set learning priorities
- Build a portfolio of projects
